# PROJECT 000

BBCP (Bare Bones Copy) - A simple utility that copies the contents of the source file to target. If target is a directory, bbcp will copy source file into this directory.

Synopsis: 
    bbcp source target

Example usage: 
    bbcp file1 file2
    bbcp file1 dir
    bbcp file1 dir/file2
    bbcp file1 dir/subdir/subsubdir/file2
